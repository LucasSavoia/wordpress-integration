## Sobre

Faça uma integração com o WordPress para levar dados de um blog para a sua loja VTEX IO e criar uma página inicial do blog, páginas de categorias e páginas de postagens do blog.

## Funcionalidades
- Página inicial, categorias, postagens do blog e componentes de busca
- Associe as postagens do blog aos produtos e vice-versa, utilizando o sistema de tags do WordPress
- Exiba o conteúdo de várias instalações do WordPress na mesma loja VTEX
